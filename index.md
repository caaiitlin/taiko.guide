# Welcome to taiko.guide

**taiko.guide** is an English-language documentation project for the Japanese rhythm game **Taiko no Tatsujin** (太鼓の達人).

Our goal is to provide clear, up-to-date documentation covering the core gameplay of Taiko no Tatsujin - specifically focusing on documenting how to play, understand, and improve at the game.

Whether you're completely new to the game or an experienced player looking for detailed reference material, **taiko.guide** aims to become a trusted reference for the English-speaking community.

Use the navigation sidebar to browse by topic, or start with one of the guides below.
